Requirements: Python 2.7, Flask, SqlAlchemy, httplib2

This is a catalog app that provides multiple categories 
of musical instruments which can be modified by the 
allowed users



You would have to run database_setup to setup data base and
datapopulate to populate data. But the files are already run
So that can be skipped

""To run the app: python catalogapp.py""


http://localhost:5000 for the website

JSON endpoint : http://localhost:5000/JSON
